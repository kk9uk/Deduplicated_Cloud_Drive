## Usage
- Build

      make

- Upload

      java DeduplicatedCloudDrive upload <min_chunk> <avg_chunk> <max_chunk> <file_to_upload>

- Download

      java DeduplicatedCloudDrive download <file_to_download> <target_path>

- Delete

      java DeduplicatedCloudDrive delete <file_to_delete>

- Clean

      make clean
